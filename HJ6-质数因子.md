## 1. 题目描述

功能：输入一个正整数，按照从大到小的顺序输出它的所有质因子（重复的也要列举）

输入描述：输入一个整数

输出描述：按照从小到大的顺序输出它的所有质数的因子，以空格隔开



示例：

```
输入：180
输出：2 2 3 3 5
```



## 2. Solution

```python
import math

n = int(input())
for i in range(2, int(math.sqrt(n))+1):
    if n % i == 0:
        # end = ' '表示以空格隔开
        print(i, end=' ')
        n //= i
# 如果上面这个for循环没能够输出，则证明这个数本身就是一个质数，直接输出即可        
if n > 2:
    print(n)
```
