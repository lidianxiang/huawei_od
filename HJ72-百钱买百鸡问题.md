## 1. 题目描述

公元五世纪，我国古代数学家张丘建在《算经》一书中提出了“百鸡问题”：鸡翁一值钱五，鸡母一值钱三，鸡雏三值钱一。百钱买百鸡，问鸡翁、鸡母、鸡雏各几何？

现要求你打印出所有花一百元买一百只鸡的方式。



示例1:

```
输入：1
输出：
    0 25 75
    4 18 78
    8 11 81
    12 4 84
```



## 2. Solution

```python
a = input()
for i in range(100):
    for j in range(100-i):
        if 5 * i + 3 * j+ (100-i-j)/3 == 100:
            print(i, j, 100-i-j)
```
